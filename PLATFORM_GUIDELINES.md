# Platform-Specific Content Guidelines

This document defines the ideal content specifications for each social media platform to help generate platform-optimized content.

## Instagram

| Aspect | Specification |
|--------|---------------|
| **Text Length** | 125-150 characters (optimal) |
| **Max Characters** | 2,200 characters |
| **Hashtags** | 20-30 hashtags (optimal) |
| **Image Ratio** | 1:1 (square), 4:5 (portrait), 1.91:1 (landscape) |
| **Image Dimensions** | 1080x1080px (square), 1080x1350px (portrait) |
| **Tone** | Visual-first, aspirational, lifestyle-focused |
| **Call-to-Action** | Link in bio, save post, share story |
| **Emoji Usage** | 2-4 emojis recommended |
| **Caption Style** | Conversational, story-telling |

## LinkedIn

| Aspect | Specification |
|--------|---------------|
| **Text Length** | 150-200 characters (optimal) |
| **Max Characters** | 3,000 characters |
| **Hashtags** | 3-5 hashtags (optimal) |
| **Image Ratio** | 1.2:1 (landscape) |
| **Image Dimensions** | 1200x627px |
| **Tone** | Professional, thought-leadership, industry insights |
| **Call-to-Action** | Comment, share, connect |
| **Emoji Usage** | 0-2 emojis (use sparingly) |
| **Caption Style** | Formal, business-focused, value-driven |

## Facebook

| Aspect | Specification |
|--------|---------------|
| **Text Length** | 200-250 characters (optimal) |
| **Max Characters** | 63,206 characters |
| **Hashtags** | 1-3 hashtags (optimal) |
| **Image Ratio** | 1.2:1 (landscape), 4:5 (portrait) |
| **Image Dimensions** | 1200x628px (landscape), 1080x1350px (portrait) |
| **Tone** | Community-focused, conversational, engaging |
| **Call-to-Action** | Like, comment, share, message |
| **Emoji Usage** | 1-3 emojis |
| **Caption Style** | Friendly, question-based, discussion-starter |

## X (Twitter)

| Aspect | Specification |
|--------|---------------|
| **Text Length** | 100-120 characters (optimal) |
| **Max Characters** | 280 characters |
| **Hashtags** | 1-2 hashtags (optimal) |
| **Image Ratio** | 16:9 (landscape), 1:1 (square) |
| **Image Dimensions** | 1024x512px (landscape), 512x512px (square) |
| **Tone** | Concise, witty, timely, conversational |
| **Call-to-Action** | Retweet, reply, like |
| **Emoji Usage** | 0-2 emojis |
| **Caption Style** | Punchy, direct, trend-aware |

## Website/Blog

| Aspect | Specification |
|--------|---------------|
| **Text Length** | 300-400 characters (preview) |
| **Full Content** | 800-1500 words (article) |
| **Hashtags** | Not applicable |
| **Image Ratio** | 16:9 (featured), 1:1 (thumbnails) |
| **Image Dimensions** | 1200x675px (featured), 600x600px (thumbnail) |
| **Tone** | Informative, SEO-optimized, detailed |
| **Call-to-Action** | Read more, subscribe, contact |
| **Emoji Usage** | Not recommended |
| **Caption Style** | SEO-friendly, structured, comprehensive |

## Key Principles

1. **Image-First**: Always place images above text in previews to match platform behavior
2. **Platform Tone**: Adapt language and style to match platform culture
3. **Hashtag Strategy**: Use platform-optimal hashtag counts
4. **Character Limits**: Respect platform limits while optimizing for engagement
5. **Visual Hierarchy**: Ensure text is readable and scannable
6. **Call-to-Action**: Include clear, platform-appropriate CTAs
7. **Consistency**: Maintain brand voice while adapting to platform norms
